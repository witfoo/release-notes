# WitFoo Parsers v2026.03.11-201411

**Release Date:** 2026-03-11

## Summary

This release introduces a new auto-generated parser (`wfad`) to handle previously unrecognized log messages observed in customer environments. The parser ingests generic log data via UDP and classifies unknown message types, expanding WitFoo's coverage to ensure no log signals are silently dropped during security signal processing.

## What's New

### Features

- **New Parser: `wfad`** — Auto-generated parser for unknown log messages detected in production customer environments.
  - **Stream:** GenericLog
  - **Message Type:** Unknown
  - **Entrypoint:** `udp_source`
  - Provides a catch-all ingestion path for log formats not yet matched by existing parsers, ensuring visibility into unclassified traffic for triage and future parser development.
