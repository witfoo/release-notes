# WitFoo Parsers v2026.02.25-002243

**Release Date:** 2026-02-25

## Summary
This release introduces a new auto-generated parser, **witfoostreamer**, designed to handle previously unrecognized log messages detected in customer environments. The parser supports generic log ingestion via UDP, covering both unknown log formats and audit log messages, ensuring broader signal coverage across deployed environments.

## What's New
### Features
- **New Parser: `witfoostreamer`** — Auto-generated empathetic parser that captures and processes previously unclassified log messages observed in production customer environments.
  - **Stream Names:** GenericLog
  - **Message Types:** Unknown, AuditLog
  - **Entrypoint:** `udp_source`
  - Enables ingestion and normalization of log data that was previously dropped or unmatched, improving overall signal visibility for security operations teams.
