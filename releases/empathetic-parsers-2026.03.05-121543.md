# WitFoo Parsers v2026.03.05-121543

**Release Date:** 2026-03-05

## Summary

This release introduces a new auto-generated parser (`wfad`) to handle previously unrecognized log messages observed in customer environments. The parser ingests generic log data via UDP and classifies unknown message types, expanding coverage and reducing unparsed log volume in production deployments.

## What's New

### Features

- **New Parser: `wfad`** — Added a new parser to capture and process previously unclassified log messages detected across customer environments.
  - **Stream:** GenericLog
  - **Message Type:** Unknown
  - **Entrypoint:** `udp_source`
  - This parser ensures that log data arriving over UDP which does not match existing fingerprints is still ingested, normalized, and available for triage rather than silently dropped.
