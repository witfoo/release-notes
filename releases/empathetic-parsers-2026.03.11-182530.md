# WitFoo Parsers v2026.03.11-182530

**Release Date:** 2026-03-11

## Summary
This release introduces a new auto-generated parser (`wfad`) to handle previously unrecognized log messages detected in customer environments. The parser ingests generic log data via UDP and classifies unknown message types, expanding WitFoo's coverage to **250+ supported log format parsers** and reducing unprocessed signal gaps in customer deployments.

## What's New
### Features
- **New Parser: `wfad`** — Auto-generated parser for unknown log messages observed in production customer environments.
  - **Stream:** GenericLog
  - **Message Type:** Unknown
  - **Entrypoint:** `udp_source`
  - Enables capture and normalization of previously unclassified log traffic arriving over UDP, ensuring these signals are available for downstream correlation and triage rather than being silently dropped.
