# WitFoo Parsers v2026.07.01-144526

**Release Date:** 2026-07-01

## Summary

This release expands WitFoo Parsers to 274 supported log formats with the addition of a new VMware vSphere SSO parser and a new Silverfort identity-protection parser. Existing VMware ESXi parsing coverage was also widened to capture additional subsystem log sources. These additions strengthen visibility across virtualization infrastructure and identity security domains.

## What's New

### Features

- **VMware vSphere SSO parser** — New parser for vCenter Security Token Service (STS) authentication logs, enabling ingestion and signal processing of vSphere Single Sign-On events
- **Silverfort identity-protection parser** — New parser for Silverfort CEF-formatted logs, providing coverage for identity-protection and adaptive authentication signals

### Improvements

- **VMware ESXi parser widened** — Extended fingerprint coverage to include `healthd` and `sandboxd` subsystem log events, improving completeness of ESXi host-level telemetry ingestion
- **Connector module updated** — Supporting changes to the connector pipeline to accommodate the new and updated parsers
