# WitFoo Parsers v2026.03.11-142529

**Release Date:** 2026-03-11

## Summary
This release introduces a new auto-generated parser (`wfad`) to handle previously unrecognized log messages observed in customer environments. The parser ingests generic log data via UDP and classifies unknown message types, expanding coverage for log formats that were previously dropped or left unprocessed in the signal pipeline.

## What's New

- **New Parser: `wfad`** — Added an auto-generated parser for unknown log messages detected in production customer environments.
  - **Stream:** GenericLog
  - **Message Type:** Unknown
  - **Entrypoint:** `udp_source`
  - Ensures that previously unclassified UDP log traffic is captured and available for triage, reducing blind spots in security signal processing.
