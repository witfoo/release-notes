# WitFoo Parsers v2026.03.10-061948

**Release Date:** 2026-03-10

## Summary
This release introduces a new auto-generated parser, `witfooad`, designed to handle previously unrecognized log messages detected in customer environments. The parser supports UDP ingestion and classifies incoming events as either `AuditLog` or `Unknown` message types, expanding coverage for log formats encountered in production deployments.

## What's New
### Features
- **New parser: `witfooad`** — Auto-generated empathetic parser for unknown log messages observed in customer environments.
  - **Stream:** GenericLog
  - **Message Types:** Unknown, AuditLog
  - **Entrypoint:** `udp_source`
  - Enables capture and classification of previously unhandled log formats arriving over UDP, reducing blind spots in security signal processing pipelines.
