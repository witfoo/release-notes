# WitFoo Parsers v2026.03.12-081729

## Summary

This release introduces a new auto-generated parser (`wfad`) to handle previously unrecognized log messages observed in customer environments. The parser ingests generic log data via UDP and classifies unknown message types, expanding coverage for environments producing log formats not yet matched by existing parsers. All quality gates—including compilation, unit tests, lint, fingerprint uniqueness, and registration—passed successfully.

## What's New

### Features

- **New Parser: `wfad`** — Auto-generated parser designed to capture and process unknown log messages detected in production customer environments.
  - **Stream:** `GenericLog`
  - **Message Type:** `Unknown`
  - **Entrypoint:** `udp_source`
  - Provides a catch-all ingestion path for log formats that do not match any of the existing 249+ parser fingerprints, ensuring no log data is silently dropped.
