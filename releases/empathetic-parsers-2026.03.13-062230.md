# WitFoo Parsers v2026.03.13-062230

## Summary
This release introduces a new auto-generated parser, `witfooad`, designed to handle previously unrecognized log messages detected in customer environments. The parser supports generic log ingestion via UDP and classifies messages into `AuditLog` and `Unknown` message types, expanding coverage for log formats encountered in production deployments.
