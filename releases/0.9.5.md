# WitFoo Analytics v0.9.5

**Release Date:** 2026-04-04

## Summary

WitFoo Analytics v0.9.5 delivers significant security hardening with 26 audit findings remediated, a major supply chain internalization effort covering 450+ Go packages and npm offline caches, and a comprehensive UI framework upgrade to Carbon Components Svelte 0.105. ## Breaking Changes

- **Carbon Components Svelte upgraded from 0.89.11 to 0.105.0** — Custom components extending Carbon `DataTableSkeleton`, modal buttons, or `TextArea` may require updates to match the new component API. E2E tests targeting modal Cancel buttons must now scope selectors to visible modals.
- **SSR disabled globally** — All pages now render client-side only. Deployments relying on server-side rendering behavior should verify compatibility.
- **Supply chain internalized** — Go dependencies are now vendored and npm packages served from offline caches via Git LFS. Build environments no longer require external registry access for declared dependencies.
