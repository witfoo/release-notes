# WitFoo Parsers v2026.05.14-070839

**Release Date:** 2026-05-14

## Summary
This release introduces a new auto-generated parser for `rpcstatd` log messages detected in customer environments. The parser enables ingestion and processing of previously unrecognized RPC status daemon logs arriving via UDP sources, expanding coverage for environments relying on NFS/RPC infrastructure monitoring.

## What's New
### Features
- **New parser: `rpcstatd`** — Adds support for parsing RPC status daemon (`rpc.statd`) log messages, commonly associated with NFS file locking and network status monitoring.
  - **Stream:** GenericLog
  - **Message Type:** Unknown (empathetic catch-all for unclassified `rpcstatd` messages)
  - **Entrypoint:** `udp_source`
  - Automatically generated in response to unrecognized log patterns observed in customer environments, ensuring these signals are captured rather than silently dropped.
