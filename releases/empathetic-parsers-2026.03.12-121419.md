# WitFoo Parsers v2026.03.12-121419

**Release Date:** 2026-03-12

## Summary
This release introduces a new auto-generated parser, `witfooad`, designed to capture and classify previously unknown log messages detected in customer environments. The parser ingests via UDP and routes unrecognized log formats into the GenericLog stream, ensuring that no log data is silently dropped during security signal processing.

## What's New
### Features
- **New Parser: `witfooad`** — Auto-generated parser for handling unknown log messages observed in production customer environments.
  - **Stream:** GenericLog
  - **Message Type:** Unknown
  - **Entrypoint:** `udp_source`
  - Ensures previously unclassified UDP log traffic is captured, normalized, and available for triage and future parser development.
