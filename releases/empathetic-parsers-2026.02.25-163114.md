# WitFoo Parsers v2026.02.25-163114

**Release Date:** 2026-02-25

## Summary

This release introduces a new **Cassandra** database log parser, expanding WitFoo's parser library to support Apache Cassandra audit and general log ingestion. The parser was auto-generated in response to previously unrecognized log messages detected in customer environments, ensuring broader signal coverage for security operations teams monitoring database infrastructure.

## What's New

### Features

- **Cassandra Log Parser** — Added a new parser for Apache Cassandra database logs:
  - **Stream:** GenericLog
  - **Message Types:** AuditLog, Unknown
  - **Entrypoint:** `udp_source`
  - Enables ingestion and classification of Cassandra audit log events, supporting database activity monitoring and security signal correlation
  - Automatically classifies unrecognized Cassandra log messages under the `Unknown` message type for triage and future refinement
