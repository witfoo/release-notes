## Summary
This release introduces a new auto-generated parser for Oracle Cloud Infrastructure (OCI) log formats. The `oracleoci` parser enables ingestion and processing of previously unrecognized log messages detected in customer environments, expanding WitFoo's coverage to 250+ supported log format parsers.

## What's New
### Features
- **New Parser: `oracleoci`** — Added support for Oracle Cloud Infrastructure log ingestion via UDP source. This parser handles generic log streams (`GenericLog`) from OCI environments that were previously categorized as unknown message types. The parser was auto-generated in response to unrecognized log traffic observed in production customer deployments, ensuring these signals are now captured and available for security analysis.
  - **Stream Names:** GenericLog
  - **Message Types:** Unknown
  - **Entrypoint:** `udp_source`
