# WitFoo Parsers v2026.03.08-181129

## Summary

This release introduces a new auto-generated parser for **MySQL** log formats, expanding WitFoo Parsers' coverage to handle previously unrecognized MySQL log messages detected in customer environments. The parser supports multiple message types including access logs and audit logs, ingested via UDP, strengthening visibility into database activity for security operations teams.

## What's New

### Features

- **MySQL Parser** — Added a new parser for MySQL log sources with the following capabilities:

  - **Message Types:** AccessLog, AuditLog, Unknown
  - **Entrypoint:** `udp_source`
  - Enables parsing and classification of MySQL database log events, supporting security monitoring use cases such as access auditing and anomaly detection
  - Addresses previously unclassified log messages observed in production customer environments
