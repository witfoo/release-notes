# WitFoo Parsers v2026.05.07-070756

**Release Date:** 2026-05-07

## Summary
This release introduces a new auto-generated parser for Qualys BPF (`qualysbpf`) log sources, expanding WitFoo's coverage to 250+ supported log formats. The parser was created in response to previously unrecognized log messages detected in customer environments, ensuring these signals are now properly ingested and classified.

## What's New
### Features
- **New Parser: `qualysbpf`** — Added support for Qualys BPF log ingestion via UDP source entrypoint. The parser handles two message types:
  - `AuditLog` — Structured audit events from Qualys BPF sources
  - `Unknown` — Catch-all classification for unrecognized Qualys BPF messages, ensuring no log data is silently dropped
- Stream classification: Logs are mapped to the `GenericLog` stream for downstream signal processing
