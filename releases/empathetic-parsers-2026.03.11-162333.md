# WitFoo Parsers v2026.03.11-162333

**Release Date:** 2026-03-11

## Summary
This release introduces a new auto-generated parser, `witfooad`, designed to capture and classify previously unknown log messages detected in customer environments. The parser ingests via UDP and routes unrecognized log data into the GenericLog stream, ensuring that no log traffic is silently dropped and enabling analysts to triage and promote these signals as needed.

## What's New
### Features
- **New Parser: `witfooad`** — Auto-generated parser for handling unknown log formats observed in production customer environments.
  - **Stream:** GenericLog
  - **Message Type:** Unknown
  - **Entrypoint:** `udp_source`
  - Provides empathetic coverage for log messages that do not yet match any of the 249+ existing parser fingerprints, preventing data loss and supporting future parser development based on real-world traffic patterns.
