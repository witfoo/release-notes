# WitFoo Parsers v2026.03.09-221228

**Release Date:** 2026-03-09

## Summary
This release introduces a new auto-generated parser, `witfooad`, designed to handle previously unrecognized log messages detected in customer environments. The parser ingests generic log data via UDP and classifies unknown message types, expanding WitFoo's coverage of real-world log formats encountered in production deployments.

## What's New
### Features
- **New Parser: `witfooad`** — Auto-generated parser for unknown log messages observed in customer environments. Configured with the following characteristics:
  - **Stream Name:** GenericLog
  - **Message Type:** Unknown
  - **Entrypoint:** `udp_source`
  - This parser captures and normalizes previously unclassified log traffic, ensuring that unrecognized messages are ingested and available for triage rather than silently dropped.
