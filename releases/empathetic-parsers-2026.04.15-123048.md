## Summary

This release introduces a new parser for **ESET** security product logs, expanding WitFoo's log format coverage to support ESET audit log ingestion via UDP. The parser was auto-generated in response to previously unrecognized log messages detected in customer environments, ensuring broader signal coverage for security operations teams relying on ESET endpoint protection.

## What's New

- **New ESET Parser** — Added support for parsing ESET security product logs with the following characteristics:

  - **Entrypoint:** `udp_source` — enables ingestion of ESET logs delivered over UDP/syslog
  - Provides regex fingerprinting and serial pipeline processing for ESET audit events, enabling correlation and signal generation from previously unclassified log data
