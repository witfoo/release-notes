# WitFoo Parsers v2026.03.12-201427

**Release Date:** 2026-03-12

## Summary
This release introduces a new auto-generated parser (`wfad`) to handle previously unrecognized log messages detected in customer environments. The parser ingests generic log data via UDP, classifying messages as either audit log entries or unknown events, expanding WitFoo's coverage to **250+ supported log format parsers**.

## What's New
### Features
- **New Parser: `wfad`** — Auto-generated parser designed to capture and classify log messages that were previously unrecognized in production customer environments.
  - **Stream:** GenericLog
  - **Message Types:** AuditLog, Unknown
  - **Entrypoint:** `udp_source`
  - Enables ingestion and normalization of audit log signals that were previously dropped or left unprocessed, improving visibility for security operations teams.
