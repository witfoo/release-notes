# WitFoo Parsers v2026.03.13-002059

**Release Date:** 2026-03-13

## Summary

This release introduces a new auto-generated parser (`wfad`) to handle previously unrecognized log formats detected in customer environments. The parser captures generic log streams and audit log messages arriving via UDP, expanding WitFoo's coverage to reduce unknown signal volume in production deployments.

## What's New

### Features

- **New Parser: `wfad`** — Auto-generated parser designed to process previously unclassified log messages observed in customer environments.
  - **Stream:** GenericLog
  - **Message Types:** Unknown, AuditLog
  - **Entrypoint:** `udp_source`
  - Enables ingestion and classification of log data that was previously dropped or categorized as unknown, improving signal visibility for security operations teams.
