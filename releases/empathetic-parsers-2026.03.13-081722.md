## Summary

This release introduces a new auto-generated parser (`wfad`) designed to capture and classify previously unknown log messages detected in customer environments. The parser processes generic log streams via UDP ingestion, routing messages into `Unknown` and `AuditLog` message types for downstream security signal processing.
